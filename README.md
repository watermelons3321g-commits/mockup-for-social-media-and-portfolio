# Gaming Community

Веб-приложение для общения и взаимодействия геймеров.

## Описание

Gaming Community — это современное веб-приложение, предназначенное для объединения игроков и создания сообщества вокруг видеоигр. Приложение предоставляет функции для авторизации, просмотра профилей, галереи, поиска друзей и участия в турнирах.

## Функциональность

- **Система авторизации** - Регистрация и вход пользователей
- **Профили пользователей** - Просмотр информации об игроках (рейтинг, винреит, основные игры)
- **Галерея** - Просмотр контента сообщества
- **Друзья** - Система управления друзьями
- **Турниры** - Информация о турнирах и соревнованиях
- **Избранное** - Сохранение любимого контента
- **Лайки** - Оценка контента
- **Подписки** - Подписка на каналы
- **Уведомления** - Система тостов для информирования пользователя

## Технологии

- **HTML5** - Структура приложения
- **CSS3** - Стилизация с темным дизайном
- **JavaScript (ES6)** - Логика приложения
- **LocalStorage** - Хранение данных пользователей

## Структура файлов

```
.
├── index.html       # Страница входа/регистрации
├── main.html        # Главная страница
├── friends.html     # Страница друзей
├── gallery.html     # Галерея контента
├── tournaments.html # Страница турниров
├── app.js           # Основная логика приложения
├── style.css        # Стили приложения
└── README.md        # Этот файл
```

## Установка и использование

1. Скачайте файлы проекта
2. Откройте `index.html` в веб-браузере
3. Создайте аккаунт или используйте учетные данные по умолчанию:
   - **Логин:** admin
   - **Пароль:** admin

## Возможности

### Аутентификация
- Регистрация новых пользователей
- Вход в аккаунт
- Выход из системы

### Взаимодействие
- Просмотр профилей других пользователей
- Отправка запросов в друзья
- Лайки контента
- Добавление в избранное
- Подписка на каналы

## Заметки

Данные хранятся в `localStorage` браузера, поэтому они сохраняются между сеансами в одном браузере.
